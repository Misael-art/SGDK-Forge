>> Como compilar:
-----------------

1- Certifique-se que vc possui o SGDK e JAVA instalado em seu computador.
   Geralmente o SGDK é colocado na pasta "C:\SGDK\"
   Configure corretamente o PATH do Windows, conforme o video do canal GameDevBoss.
   link do vídeo: https://youtu.be/H0XNUe4wY7E

2- Coloque o conteudo deste projeto dentro da pasta "C:\GENDK\HAMOOPIG\"

3- Dentro da subpasta Out "C:\GENDK\HAMOOPIG\out" basta clicar nos atalhos .BAT para compilar!
   (A) "_COMPILAR" -> use este caso tenha feito apenas alteracoes no codigo 
   (B) "_COMPILAR_E_ATUALIZAR" -> use este caso tenha alterado imagens, sons e etc.

4- A rom estara dentro desta pasta, com o nome [rom.bin]

5- Eu estou usando o emulador BIZHAWK para executar o jogo apos a compilacao, no meu caso,
   ele está instalado na pasta "C:\GAMES\Bizhawk\". Caso queira usar outro emulador, altere
   o arquivo .BAT, para ele carregar o emulador desejado.
