# Referencias Historicas

Esta pasta preserva materiais originais e historicos do projeto que ajudam a
entender sua origem, autoria e fluxo legado, mas que **nao fazem parte** do
worktree operacional do SGDK.

## O que fica aqui

- PDFs e documentos tecnicos originais
- readmes herdados de fluxos antigos
- notas que mencionam caminhos legados como `C:\\GENDK\\...`

## Regra importante

Esses arquivos ficam fora de `KOF94/` para que o SGDK root real permaneça
limpo e focado apenas em `src/`, `res/`, `inc/` e `out/`.
