>> How to compile:
-----------------

1- Make sure you have the SGDK and JAVA installed on your computer.
   Usually the SGDK is placed in the folder "C:\SGDK\"
   Correctly configure the Windows PATH, as shown in the GameDevBoss channel video.
   Video link: https://youtu.be/H0XNUe4wY7E

2- Place the contents of this project inside the folder "C:\GENDK\HAMOOPIG\"

3- Inside the Out subfolder "C:\GENDK\HAMOOPIG\out" just click on the .BAT shortcuts to compile!
   (A) "_COMPILAR" -> use this if you only made changes to the code
   (B) "_COMPILAR_E_UPDATE" -> use this if you have changed images, sounds and etc.

4- The rom will be inside this folder, with the name [rom.bin]

5- I am using BIZHAWK emulator to run the game after compilation, in my case,
   it is installed in the folder "C:\GAMES\Bizhawk\". If you want to use another emulator, change
   the .BAT file, for it to load the desired emulator.
