# MegaDriving [VER.1.0] [SGDK 211] [GEN] [ENGINE] [CORRIDA]

Protótipo de simulador de corrida explorando técnicas de Pseudo-3D e Road Scrolling.

## Diferenciais Técnicos
- **Road Scrolling**: Implementação clássica de estrada usando efeitos de interrupção horizontal (H-Int).
- **Efeito Mode 7 Fake**: Simulação de profundidade e perspectiva sem hardware dedicado.
- **Escala de Sprites**: Sistema de escalonamento dinâmico para objetos na estrada.

## Estrutura
Contém exemplos de como manipular os registros do VDP durante o scanline para criar a ilusão de profundidade.
