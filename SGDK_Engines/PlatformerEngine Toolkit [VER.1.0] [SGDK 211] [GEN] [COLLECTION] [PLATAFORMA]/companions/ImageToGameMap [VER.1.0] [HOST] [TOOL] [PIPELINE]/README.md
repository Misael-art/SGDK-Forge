# ImageToGameMap [VER.1.0] [HOST] [TOOL] [PIPELINE]

Esta e a entrada canonica da ferramenta auxiliar `ImageToGameMap`.
Ela nao e uma ROM de Mega Drive e, por isso, o wrapper a trata como pacote
pedagogico com build desativado.

## Onde o conteudo real ficou

- `../../upstream/ImageToGameMap`

## Para que serve

- estudar o pipeline de conversao de mapas
- entender como a engine de plataforma consome dados externos
- separar claramente runtime SGDK de ferramentas de host

## Regra pratica

- use esta pasta para leitura e contexto
- abra `../../upstream/ImageToGameMap` em um ambiente Unity quando quiser
  explorar a ferramenta em si
- volte para a raiz da colecao para entender o fluxo completo
