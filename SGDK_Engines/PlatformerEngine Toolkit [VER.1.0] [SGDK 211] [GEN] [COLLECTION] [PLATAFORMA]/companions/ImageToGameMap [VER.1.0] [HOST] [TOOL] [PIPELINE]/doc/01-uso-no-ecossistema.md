# Uso no Ecossistema

`ImageToGameMap` ensina uma liccao importante para iniciantes: um projeto de
jogo serio raramente vive apenas no codigo do console. Ferramentas de pipeline
tambem fazem parte da arquitetura e merecem uma apresentacao organizada.

Neste pacote:

- a ferramenta fica em `companions/`
- o conteudo original fica em `upstream/ImageToGameMap`
- a engine buildavel fica em `variants/PlatformerEngine Core ...`
