# Engine Core

O item realmente compilavel deste pacote e:

- `variants/PlatformerEngine Core [VER.1.0] [SGDK 211] [GEN] [ENGINE] [PLATAFORMA]`

## Onde o codigo real ficou

O SGDK root real foi preservado em:

- `upstream/PlatformerEngine`

## Por que usar uma variante canonica

- a raiz da variante oferece onboarding limpo
- os wrappers funcionam sem depender dos scripts originais do upstream
- o desenvolvedor iniciante sabe exatamente onde compilar e onde editar

## Onde editar

- codigo C: `upstream/PlatformerEngine/src/`
- headers: `upstream/PlatformerEngine/inc/`
- recursos: `upstream/PlatformerEngine/res/`
- dados de exemplo: `upstream/Level/`
