# Fluxo Pedagogico

## Caminho recomendado para iniciantes

1. Leia a documentacao desta colecao.
2. Abra a variante `PlatformerEngine Core`.
3. Rode `build.bat` pela variante para gerar a ROM.
4. Inspecione `upstream/Level` para entender os dados de fase.
5. Estude `companions/ImageToGameMap ...` para entender como os mapas podem ser
   preparados fora do console.

## O que este pacote ensina

- como separar engine, ferramenta e assets sem bagunca
- como usar um wrapper central para proteger builds em estruturas diferentes
- como preservar o upstream sem sacrificar a didatica do workspace
