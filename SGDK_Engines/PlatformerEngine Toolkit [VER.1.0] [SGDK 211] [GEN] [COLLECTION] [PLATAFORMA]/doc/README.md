# Documentacao da Colecao

Esta pasta concentra o contexto pedagogico do `PlatformerEngine Toolkit`.

1. `01-visao-geral.md`
2. `02-organizacao-da-colecao.md`

## Objetivo desta documentacao

- explicar por que esta raiz e uma colecao e nao uma `ENGINE` unica
- mostrar onde a parte SGDK buildavel realmente vive
- orientar iniciantes sobre o fluxo entre mapas, engine e recursos
- preservar a leitura do upstream sem poluir a superficie canonica
