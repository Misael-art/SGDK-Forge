# PlatformerEngine [VER.1.0] [SGDK 211] [GEN] [ENGINE] [PLATAFORMA]

Este motor de plataforma foi desenvolvido por Gerard Gascon e oferece uma base sólida para jogos de ação 2D no Mega Drive usando SGDK.

## Diferenciais Técnicos
- **Integração com LDtk**: Suporte nativo para mapas criados no editor LDtk.
- **Física Avançada**: Implementação de Coyote Time e Jump Buffer para um gameplay mais fluido.
- **Gerenciamento de Colisões**: Sistema otimizado para lidar com múltiplos sprites e tiles.
- **Compatibilidade**: Testado com SGDK 1.60 e superiores (Recomendado 2.11).

## Estrutura
- `src/`: Código fonte principal em C.
- `res/`: Recursos de tileset e sprites.
- `inc/`: Headers e definições.

## Como Usar
1. Configure seu ambiente SGDK.
2. Compile usando `make` ou os wrappers do projeto MegaDrive_DEV.
3. Edite os mapas no LDtk para criar suas fases.
