# Relacao com a Colecao

Esta variante nao deve duplicar:

- a documentacao geral da colecao
- a ferramenta `ImageToGameMap`
- os dados de nivel preservados em `upstream/Level`

Use esta variante quando o foco for compilar, depurar e evoluir a engine em si.
Volte para a raiz da colecao quando o assunto for organizacao, nomenclatura e
pipeline pedagogico.
