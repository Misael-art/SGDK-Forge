# 🚀 Quick Start - 5 minutos

## Pré-requisitos (✅ já feito no workspace)
- ✅ SGDK 2.11 configurado
- ✅ Java instalado
- ✅ Emulador (BlastEm em `tools/emuladores/`)

## Passo 1: Escolha seu nível
```
Nunca fez jogo de plataforma? → core/
Tem experiência? → standard/
Quer otimizar? → advanced/
```

## Passo 2: Abra a pasta
Windows:
```cmd
cd SGDK_Engines\PlatformerEngine_CONSOLIDATED [VER.1.0] [SGDK 211] [GEN] [COLLECTION] [PLATAFORMA]\standard
```

## Passo 3: Compile
```cmd
build.bat
```

Esperado: `rom.bin` gerado em `out/rom.bin` ✅

## Passo 4: Teste no Emulador
```cmd
run.bat
```

Esperado: Emulador abre com jogo rodando a 60fps ✅

## Passo 5: Explore o Código
Abra `src/main.c` e leia os comentários.

## 🎯 Próximo Passo
- Se funcionou → Leia `doc/01_ARCHITECTURE.md`
- Se deu erro → Leia `TROUBLESHOOTING.md`

---

**Parabéns! 🎉 Você está rodando um jogo de plataforma no Mega Drive.**
