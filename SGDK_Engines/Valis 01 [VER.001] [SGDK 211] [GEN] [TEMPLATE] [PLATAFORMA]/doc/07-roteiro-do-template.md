# Roteiro do Template

Este template foi montado para mostrar um ciclo bem pequeno e bem legivel:

1. fundo em `res/gfx.res`
2. sprites em `res/sprites.res`
3. logica em `src/main.c`
4. build pelo wrapper central

## Sugestoes de exercicio

1. Troque `bga.png` por outro fundo de teste.
2. Ajuste as posicoes iniciais dos sprites em `src/main.c`.
3. Adicione um movimento simples usando `JOY_readJoypad`.
4. Crie um quarto sprite no `sprites.res` e replique o padrao do exemplo.
