# XGM2 Driver Samples [VER.001] [SGDK 211] [GEN] [COLLECTION] [AUDIO]

Esta colecao canonica agrega exemplos oficiais de `XGM2` preservados a partir
do SGDK 2.11 de Stephane Dallongeville.

## O que existe aqui

- `variants/XGM2 Sound Test ...`: painel completo para estudar drivers de som
- `variants/XGM2 Sonic Audio Demo ...`: exemplo de jogo usando `XGM2` com PCM
- `upstream/`: snapshot dos samples oficiais usados como base

## Como usar

1. Leia `doc/README.md`.
2. Escolha uma variante em `variants/`.
3. Rode `build.bat` e `run.bat` pela variante escolhida.

## Fonte oficial

- Repositorio oficial: `https://github.com/Stephane-D/SGDK`
- Samples preservados a partir de `sample/snd/sound-test` e `sample/game/sonic`
