# imgs-example [VER.001] [SGDK 211] [GEN] [ENGINE] [GRAFICOS]

Este projeto agora funciona como um **exemplo pedagogico limpo** para estudar
fundo, sprite e input basico com SGDK. O foco e dar a um iniciante um fluxo
claro de:

- carregar um background
- carregar um sprite
- mover o sprite com o direcional
- testar pequenas interacoes visuais

## Controles

- Direcional: move o personagem
- `A`: espelha horizontalmente o sprite
- `B`: reposiciona no centro
