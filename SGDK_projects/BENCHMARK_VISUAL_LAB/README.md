# BENCHMARK_VISUAL_LAB

Laboratorio visual canonico para provar em ROM a diferenca entre uma traducao `basic` e uma traducao `elite`.

## Missao

- exibir comparativos lado a lado em 320x224
- fechar o ciclo `curadoria -> ROM -> BlastEm`
- servir como gabarito visual para futuras heuristicas

## Caso ativo

- `verdant_forest_depth_scene`
- `basic_score = 0.6577`
- `elite_score = 0.8419`
- `delta = 0.1842`

## Controles

- `A`: liga/desliga overlay textual
- `B`: liga/desliga `BG_A`
- `C`: liga/desliga `BG_B`

## Regra de ouro

Se nao foi visto no BlastEm, ainda nao virou verdade operacional.
