## High-Level Vision

Projeto-laboratorio para canonizar heuristicas de excelencia visual no workspace MegaDrive_DEV.

Ele existe para provar, em ROM, que uma regra nova da skill `visual-excellence-standards` realmente melhora leitura, contraste de planos e clareza de animacao em hardware-alvo.

## Quick Start

1. `build.bat` para compilar.
2. `run.bat` para abrir a ROM.
3. Use `LEFT` e `RIGHT` para alternar entre as provas.
