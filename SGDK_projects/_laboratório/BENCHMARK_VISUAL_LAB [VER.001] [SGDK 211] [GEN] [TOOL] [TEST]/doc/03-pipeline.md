# 03 - Pipeline

## Pipeline canonico de uma nova heuristica

1. observar o sintoma em um projeto real
2. registrar no `doc/03_art/02_visual_feedback_bank.md` do workspace
3. atualizar `visual-excellence-standards`
4. medir com `tools/image-tools/analyze_aesthetic.py`
5. integrar ao `validation_report.json`
6. provar em ROM no `BENCHMARK_VISUAL_LAB`
7. validar em BlastEm

## Regra de ouro

Nenhuma heuristica visual vira regra so por convencimento verbal.

Se nao foi vista no laboratorio em ROM, ela ainda nao foi canonizada.
