# 11 - Game Design Document — BENCHMARK_VISUAL_LAB [VER.001] [SGDK 211] [GEN] [TOOL] [TEST]

## Visao

Este projeto e um laboratorio de validacao visual para o workspace MegaDrive_DEV. Ele nao busca entretenimento final, mas sim oferecer uma ROM curta e repetivel onde novas heuristicas de arte podem ser comparadas e aprovadas.

## Mecanicas core

- alternar entre provas fixas do laboratorio
- exibir criterio de leitura da prova atual
- servir de arena para futuras comparacoes A/B de assets

## Progressao

- o usuario percorre os modos do laboratorio
- cada modo prova uma dimensao especifica da heuristica visual

## Regras e limites

- sem escopo narrativo ou de gameplay final
- toda mudanca visual deve refletir a skill mestra do workspace
- toda regra nova precisa de prova em BlastEm antes de ser canonizada

## Escopo atual

- fase atual: fundacao estrutural do laboratorio
- fora do escopo desta fase: pacote final de assets e baseline de capturas
