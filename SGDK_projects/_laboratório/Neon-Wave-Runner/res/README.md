# Recursos do Modelo

Este projeto adota o pipeline canonico:

- `res/data/`: entrada bruta;
- `res/data/backup/`: backups automaticos;
- `res/`: saida final pronta para o SGDK.

Arquivos de exemplo:
- use `resources.res.example` como referencia de declaracao;
- renomeie para `resources.res` quando tiver assets reais.
