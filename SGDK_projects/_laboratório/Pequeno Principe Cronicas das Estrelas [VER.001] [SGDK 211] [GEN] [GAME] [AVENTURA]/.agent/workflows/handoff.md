# Workflow: Handoff

Use este fluxo ao encerrar uma sessao relevante.

1. Resuma o que mudou de fato.
2. Registre o que continua placeholder ou parcial.
3. Atualize o documento de estado operacional quando existir.
4. Declare o que foi buildado e o que ainda nao foi testado.
5. Se houver deriva documental, nao a esconda.
