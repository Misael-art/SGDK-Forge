## B-612 — PLANETA DE ORIGEM

### Entrada
```
CODEX — B-612
"Um planeta tão pequeno que
 o pôr do sol pode ser visto
 quarenta e três vezes por dia
 apenas movendo a cadeira."
```

### Rosa — Primeiro encontro
```
ROSA:
"Você foi embora sem me dizer
 que me amava.
 Eu sabia, mas precisava
 ouvir."

ROSA:
"Cuide das minhas espinhas.
 Elas são inúteis — eu sei.
 Mas eu gosto delas
 assim mesmo."
```

### Rosa — Ao regar
```
ROSA:
"Obrigada.
 Água fria seria melhor,
 mas esta serve.
 Você sempre foi assim."
```

### Rosa — Despedida
```
ROSA:
"Vá.
 Você já foi antes.
 Mas desta vez
 volte."
```
