# Visao Geral - SGDK-Forge

## Intencao

SGDK-Forge e a versao publica desta base de trabalho: um repositorio demonstrativo para pipelines automatizados de assets em SGDK 2.11.

Slogan:

`Modernizing the Sega Genesis Asset Pipeline`

## Pilares

- `data/` como origem canonica dos assets brutos.
- `res/` como saida final compativel com o SGDK.
- validacao preventiva para evitar falhas classicas de paleta e transparencia.
- documentacao clara para uso pela comunidade.

## Escopo

Este projeto existe para mostrar como estruturar um pipeline moderno de assets para Sega Genesis, com auto-slicing, validacao de paleta e build resiliente via `sgdk_wrapper`.

## Roadmap

1. consolidar a documentacao publica do repositorio;
2. ampliar os exemplos de assets e casos de teste;
3. continuar evoluindo o wrapper como referencia canonica de ingestao e correcao de imagens.
