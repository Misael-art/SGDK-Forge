# Overview - SGDK-Forge

## Intent

SGDK-Forge is the public-facing version of this workbench: a demonstrative repository for automated SGDK 2.11 asset pipelines.

Slogan:

`Modernizing the Sega Genesis Asset Pipeline`

## Pillars

- `data/` as the canonical source of raw assets.
- `res/` as the final SGDK-compatible output area.
- preventative validation to avoid classic palette and transparency failures.
- clear documentation aimed at community adoption and reuse.

## Scope

This project exists to show how to structure a modern Sega Genesis asset pipeline with auto-slicing, palette validation, and resilient builds through `sgdk_wrapper`.

## Roadmap

1. consolidate the public repository documentation;
2. expand the sample assets and regression cases;
3. continue evolving the wrapper as the canonical reference for asset ingestion and image correction.
