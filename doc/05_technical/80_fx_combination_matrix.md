# FX COMBINATION MATRIX

Nenhum FX pode existir isolado. Eles devem ser combinados para gerar o espetáculo prometido pelo nível AAA (Treasure, Konami).

## Combinações Obrigatórias

### Chuva
- + palette escura / controlada (flash de raios alterando bg)
- + reflexo no chão (água / poças)
- + vento (line scroll variando na direção da chuva)
- + lightning eventual (swaps rápidos de cores)

### Fogo
- + partículas (brasas subindo)
- + heat distortion (raster effect/line scroll na base das chamas ou bg)
- + glow / palette cycling pulsante
- + sombra dinâmica suprimida pelos focos de luz

### Vento
- + folhas, detritos ou partículas passando
- + poeira passando nos planos em diferentes velocidades
- + pano, árvores, elementos reagindo dinamicamente pela velocidade

## Regra de Ouro
Se você propor um efeito isolado, a cena será rejeitada. Todo efeito principal DEVE ancorar dois a quatro efeitos secundários que dão contexto e brutalidade à cena.
