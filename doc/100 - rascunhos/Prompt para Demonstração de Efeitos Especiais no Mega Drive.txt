Prompt para Demonstração de Efeitos Especiais no Mega Drive

Crie um programa para o Mega Drive que apresente um menu de seleção permitindo ao usuário explorar demonstrações de todos os efeitos especiais possíveis no console. O menu deve incluir opções para:





Scroll Parallax: Uma cena com múltiplas camadas de fundo movendo-se a diferentes velocidades, criando uma sensação de profundidade. O usuário pode mover a câmera para ajustar a visualização.



Sprites Animados: Um personagem ou objeto que se move e anima de forma suave, com controles para alterar a direção ou a velocidade da animação.



Efeitos de Transparência: Uma demonstração de como a transparência pode ser simulada usando técnicas como dithering, com opções para ativar/desativar o efeito e comparar o resultado.



Efeitos de Luz e Sombra: Uma cena que utiliza paletas de cores para criar ilusões de iluminação dinâmica, permitindo ao usuário ajustar a intensidade ou a posição da "fonte de luz".



Animações de Fundo: Um cenário onde elementos do fundo se movem ou animam independentemente, com controles para pausar ou alterar a velocidade das animações.



Efeitos Sonoros: Uma seção que reproduz diferentes tipos de sons, incluindo música, efeitos sonoros e vozes digitalizadas, com opções para selecionar e combinar os sons.



Simulação de Modo 7: Uma tentativa de simular efeitos de rotação e escala de planos de fundo, mesmo que não seja nativo no Mega Drive, permitindo ao usuário girar ou ampliar o cenário.

Requisitos Adicionais:





Cada opção deve levar a uma demonstração interativa onde o usuário pode controlar aspectos do efeito (ex.: mover a câmera, ajustar animações ou sons).



Inclua explicações textuais breves exibidas na tela, descrevendo como cada efeito é implementado tecnicamente no hardware do Mega Drive.



O programa deve ser intuitivo e fácil de navegar, com uma interface de menu clara e a possibilidade de retornar ao menu principal a qualquer momento.

Este programa deve destacar as capacidades gráficas e sonoras do Mega Drive, educando o usuário sobre como os efeitos são criados e explorando as limitações do console de forma prática e envolvente.
