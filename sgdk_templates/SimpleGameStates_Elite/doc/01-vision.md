# Visão de Elite: SimpleGameStates

## Objetivo
Demonstrar uma arquitetura de Maquina de Estados Finitos (FSM) limpa e expansível para o Mega Drive.

## Benchmarks
- [x] Troca de estados instantânea via função de ponteiro.
- [x] Estrutura modular (Rooms separadas).
- [ ] Facilidade para adição de novos estados (ex: Pause, Game Over).

## Metodologia
Este template foca na **Lógica de Sistemas**. Ele deve ser usado como base para RPGs, Adventures ou qualquer jogo que exija navegação complexa entre telas.
