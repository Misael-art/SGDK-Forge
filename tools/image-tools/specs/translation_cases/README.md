# Translation Cases

Este diretorio guarda os manifestos canonicos de curadoria da skill `art-translation-to-vdp`.

Cada manifesto representa um caso `basic` vs `elite` e deve apontar para:

- a imagem-fonte
- as variantes traduzidas
- o contrato da alma visual
- o delta minimo esperado

Use `../art_translation_case.template.json` como base.
