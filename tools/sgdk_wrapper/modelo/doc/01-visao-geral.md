# 01 - Visao Geral

Objetivo deste modelo:
- fornecer uma ROM inicial compilavel;
- demonstrar divisao por camadas (`core`, `system`, `scenes`);
- reduzir friccao para novos projetos;
- tornar o pipeline de assets autoexplicativo.

Estado inicial:
- cena de boot;
- menu textual;
- cena demo com atualizacao em tempo real;
- HUD de depuracao opcional.
